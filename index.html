💾 Data Backup
Export all school data and import it on any device to keep everything in sync
📤
Export Data
Download all school data as a file

This will download a .json file containing all your teachers, students, classes, marks, quizzes and settings. Use this file to restore data on any device.

📥
Import Data
Restore data from a backup file

Select a backup .json file previously exported from this app. All existing data will be replaced with the imported data.

📋 How to use Data Backup
1On your main computer — Go to Data Backup → Click "Download Backup File". A file called edulearn-backup.json will be saved.
2Share the file — Send it via WhatsApp, Google Drive, Email, or USB to the other device.
3On the other device — Open the app, login as Admin → Data Backup → Click "Choose Backup File" → select the .json file → all data loads instantly.
✓Keep it updated — Export a fresh backup whenever you make important changes (new students, marks, etc).
